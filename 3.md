# 基于比较的排序算法

|          | 时间复杂度 | 空间复杂度 | 稳定性 |
| -------- | ---------- | ---------- | ------ |
| 选择排序 | O(N^2)     | O(1)       | ❌     |
| 冒泡排序 | O(N^2)     | O(1)       | ✅     |
| 插入排序 | O(N^2)     | O(1)       | ✅     |
| 归并排序 | O(N\*logN) | O(N)       | ✅     |
| 快速排序 | O(N\*logN) | O(logN)    | ❌     |
| 堆排序   | O(N\*logN) | O(1)       | ❌     |

总结

1. 大部分时候，直接选择快速排序即可。
2. 需要稳定性的时候，选择归并排序。
3. 当空间有限的时候，选择堆排序。
